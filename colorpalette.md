# VIXI — Color Palette

#0B1F3B  
Primary Brand Color — Azul profundo utilizado para transmitir confianza médica, ciencia, seguridad y profesionalismo. Ideal para encabezados, navbar, secciones institucionales y elementos principales de marca.

#1E3A5F  
Secondary Brand Color — Azul complementario usado para variaciones del color principal, fondos secundarios, tarjetas informativas y componentes UI que requieran jerarquía visual sin perder la identidad médica.

#F4A6B7  
Accent Color — Rosa principal de acento que representa cercanía, humanidad, fertilidad y sensibilidad emocional. Se usa en botones principales, highlights, iconografía y elementos de interacción.

#F7D6DE  
Soft Accent Color — Rosa suave utilizado para fondos sutiles, secciones emocionales o humanas, bloques de contenido y áreas que buscan transmitir calidez y empatía.

#F5F7FA  
Neutral Background — Gris/blanco frío muy claro utilizado como fondo principal de contenido para mantener una estética limpia, médica y moderna.

#FFFFFF  
Base Neutral — Blanco puro utilizado para fondo general, contraste, legibilidad y composición minimalista del sitio.
